# Rectangle
## Description
File with functions for work with rectangles

## Functions:

---
### area(a, b)
Calculate area for rectangle

**Variables**:

a (float): first side

b (float): second side

---
### perimeter(r)
Calculate perimeter for rectangle

**Variables**:

a (float): first side

b (float): second side
